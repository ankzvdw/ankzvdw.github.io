⚙️ One-time Firebase setup — read this first To enable real-time sync across all browsers, follow these steps:
  1. Go to console.firebase.google.com and create a project (any name; you can disable Google Analytics).
  2. In the left sidebar choose Build → Realtime Database → Create database. Pick any region, then select Start in test mode.
  3. After it's created, copy the database URL shown at the top — it looks like https://your-project-default-rtdb.europe-west1.firebasedatabase.app
  4. Open this HTML file in any text editor, find FIREBASE_URL = 'https://console.firebase.google.com/u/0/project/deskapp-c583d/database/deskapp-c583d-default-rtdb/data' near the top of the script, and paste the URL between the quotes.
  5. Save and open in your browser. The dot top-left turns green when live.
